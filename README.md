Please report vulnerabilities privately via email:
**Email:** diyara.prestige@gmail.com

1. We will acknowledge your report within 48 hours.
2. We will provide an estimated timeline for a fix.
3. We request that you do not disclose the vulnerability publicly until a patch has been released.
